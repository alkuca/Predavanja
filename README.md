## how to run
in the root folder:
<br/><b>npm install -g truffle</b>
<br/><b>truffle develop</b>
<br/><b>migrate --reset</b>
<br/>
<br/>
open new terminal and cd into /client <br/>
<b>npm install</b>
<br/>
<b>npm run serve</b>
